# 조합 구하기
### 문제
1부터 N까지 번호가 적힌 구슬이 있습니다. 이 중 M개를 뽑는 방법의 수를 출력하는 프로그램을 작성하세요.

[입력] <br>
첫 번째 줄에 자연수 N(3<=N<=10)과 M(2<=M<=N) 이 주어집니다. <br>

[출력] <br>
첫 번째 줄에 결과를 출력합니다. 출력순서는 사전순으로 오름차순으로 출력합니다.

### 풀이
(시작 번호를 for문의 i로 지정한다.)
```java
public static void DFS(int L, int s) {
    if(L == m) {
        for (int i : out) {
            System.out.print(i + " ");
        }
        System.out.println();
    } else {
        for(int i = s; i <= n; i++) {
            out[L] = i;
            DFS(L + 1, i + 1);
        }
    }
}
```
